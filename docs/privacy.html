Privacy Policy

Last updated: May 2026

This Privacy Policy describes how Spritesheet Forge MCP ("the Service") collects, uses, and stores information when you connect and use the Service through GitHub OAuth or an MCP-compatible client.

1. Information We Collect

The Service collects the minimum information required to operate:

  • GitHub user ID — a numeric identifier provided by GitHub OAuth, used to associate your session and usage quota.
  • GitHub username (login) — stored alongside your session token for display and logging purposes.
  • Usage count — the number of tool operations you have performed in the current billing period, used to enforce quota limits.
  • Uploaded and processed files — image files you submit for processing (PNG, GIF, WebP). These are stored temporarily and deleted automatically.

We do not collect email addresses, IP addresses, or any personal information beyond what GitHub OAuth provides.

2. How We Use Your Information

  • Authentication — your GitHub user ID is used to identify your session and verify your access token.
  • Quota enforcement — your usage count is checked and incremented on each tool call to enforce the monthly operation limit.
  • File processing — uploaded files are processed by the image processing backend and the output is stored temporarily for you to retrieve.

We do not use your information for advertising, profiling, or any purpose beyond operating the Service.

3. Data Retention

  • Session tokens — expire after 30 days of inactivity.
  • Output files — all processed output files and uploaded files are automatically deleted 1 hour after creation.
  • Usage counters — reset on the 1st of each calendar month.

4. Third-Party Services

The Service relies on the following third-party infrastructure:

  • GitHub OAuth — used for authentication. Subject to GitHub's Privacy Statement.
  • Cloudflare Workers, KV, and R2 — used for hosting, session storage, quota storage, and temporary file storage. Subject to Cloudflare's Privacy Policy.
  • Google Cloud Run — used for image processing. Uploaded files are transmitted to Cloud Run for processing and are not retained beyond the duration of the request. Subject to Google Cloud's Privacy Notice.

We do not sell or share your data with any third parties beyond the infrastructure providers listed above.

5. Security

Access tokens are stored in Cloudflare KV and are not exposed in logs or responses. All communication between the client and the Service occurs over HTTPS. File storage uses Cloudflare R2 with server-side encryption.

6. Your Rights

You may revoke the Service's access to your GitHub account at any time via GitHub → Settings → Applications → Authorized OAuth Apps. Revoking access invalidates your session token and stops any further data collection.

7. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page will reflect any changes. Continued use of the Service after changes constitutes acceptance of the updated policy.

8. Contact

For privacy-related questions, please open an issue at github.com/LAXY9887/Game-Dev.-Spritesheet-Forge/issues.
